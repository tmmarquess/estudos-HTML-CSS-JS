Texto Marcado:

podemos criar também um texto marcado usando a tag Mark

Texto excluído:

podemos marcar um texto como inserido para dar a ênfase de que ele foi adicionado depois usando a tag ins (Semântica)

também é possível sublinhar usando a tag U (Não Semântica)
